Schwa in Goethes Briefen¶

Arbeitsschritte¶

  1. Texte aus der Digitalen Bibliothek exportieren (mittels Digitale Bibliothek 5)
    • Die Briefe sind in der Digitalen Bibliothek schon nach Jahr gruppiert, jedes Jahr kann einfach als Textdatei mit allen Briefen exportiert werden (kein Parsing der Briefköpfe nötig)
  2. Textdateien von ISO 8859-1 in Unicode (UTF-8) umwandeln
  3. Rechtliche Hinweise aus den Textdateien entfernen (die ersten 14 Zeilen streichen)
  4. In NLTK als Korpus einlesen, Dateiname enthält die Jahreszahl (diese Jahrezahl wird als Kategorie und Bedingung in den Berechnungen einbezogen)
  5. Frequenzlisten erstellen

        text = fileStream.read()
        fileNew = open('data/' + file, 'w', encoding="UTF-8")
        fileNew.write(text)

Schritt 3: Rechtliche Hinweise aus den Textdateien entfernen (die ersten 14 Zeilen streichen)¶

In [12]:
import os

for file in os.listdir(corpusDir):
    if file.endswith(".txt"):
        fileStream = open('data/' + file, encoding="UTF-8")
        text = fileStream.readlines()
        textStripped = text[14:]
        fileNew = open('data/stripped/' + file, 'w', encoding="UTF-8")
        fileNew.write(''.join(textStripped))

Bonus: Relative Frequenzen¶

Berechnet die relative Frequenz pro Jahr:

= Frequenz des Suchbegriffs / Anzahl Tokens im aktuellen Jahr * 100

In [114]:
def getRelativeFrequencies(cfd, conditions, samples):
    """ Nimmt eine ConditionalFrequenyList und berechnet davon Relativfreqeunzen in Prozent """
    import pandas
    import matplotlib.pyplot as plt
    data = []
    for condition in conditions:
        data.append([cfd[condition].freq(w) * 100 for w in samples])
    df = pandas.DataFrame(data, conditions, samples)
    df.plot(title="Frequenzverteilung: Relative Frequnezen (Prozent)")
    plt.show()
